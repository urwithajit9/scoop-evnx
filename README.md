# scoop-evnx

A [Scoop](https://scoop.sh/) bucket for **evnx**, the CLI tool for managing `.env` files with ease.

## 🚀 What is evnx?

`evnx` is a powerful CLI utility designed for:
* **Validation:** Ensure your environment variables meet your requirements.
* **Secret Scanning:** Prevent sensitive leaks before they happen.
* **Format Conversion:** Move between different environment formats seamlessly.
* **And More ...** Check on evnx.dev [https://evnx.dev]

---

## 📦 Installation

To install **evnx** using Scoop, run the following commands in your terminal:

### 1. Add the bucket

```bash
scoop bucket add evnx https://github.com/urwithajit9/scoop-evnx
```

### 2. Install the package

```bash
scoop install evnx
```

-----

## 🛠 Usage

Once installed, you can verify the installation and see available commands by running:

```powershell

evnx --version

# v0.3.7 or ...recent

evnx --help

```

For detailed documentation, visit [evnx.dev/guides](https://www.evnx.dev/guides).

## 🔄 Updates

To keep **evnx** up to date, simply run:

```powershell
scoop update evnx
```
